<task type="auto">
  <name>Task 2: Style in-post links as visually distinguishable</name>
  <files>_sass/custom.scss</files>
  <action>
Add a new rule block for `.post-content a` after the existing `.post-content blockquote` block (after line 419). This targets only links within blog post body text, not nav links or card links.

```css
.post-content a {
    color: var(--accent);
    text-decoration: underline;
    text-decoration-color: var(--accent-dim);
    text-underline-offset: 2px;
    transition: color 0.15s ease, text-decoration-color 0.15s ease;
}

.post-content a:hover {
    color: var(--text-primary);
    text-decoration-color: var(--text-primary);
}
```

This gives in-post links:
- Accent purple color (distinct from surrounding white body text)
- Underline in a slightly dimmer accent (subtle but clear)
- On hover: text goes white, underline follows

Do NOT modify the global `body a` rules (lines 61-71). Those stay as-is for nav links, card links, etc. The `.post-content a` selector has higher specificity (0,0,1,1 vs 0,0,0,1) and will override within post content.

Also add the same treatment for `.about-content a` so about page links are also visible:

```css
.about-content a {
    color: var(--accent);
    text-decoration: underline;
    text-decoration-color: var(--accent-dim);
    text-underline-offset: 2px;
    transition: color 0.15s ease, text-decoration-color 0.15s ease;
}

.about-content a:hover {
    color: var(--text-primary);
    text-decoration-color: var(--text-primary);
}
```
